说说JOLT

发表于 2022-08-20 分类于 Java
本文字数： 21k

JOLT 全称 JSON Language for Transform，是一个用 Java 编写的 Json 到 Json 的转换库。

概览

用户可以自行编写一个 Specification 来制定如何转换 Json，Jolt 就是根据用户编写的 Spec 来将输入 Json 转换成用户想要的输出 Json。Spec 中的转换可能不止一个，可以将一组转换链接（chained）在一起，让输入 Json 像经过流水线加工一样，最终形成输出 Json。

“加工”的重点是转换 Json 的数据结构，而不是操作特定的值。

Spec，即 Specification，用来指引 Jolt 如何进行转换。
LHS，即 Left Hand Side，代指 Spec Json 的 key。
RHS，即 Right Hand Side，代指 Spec Json 的 value。
在线测试工具：http://jolt-demo.appspot.com)

cardinality

cardinality模式的作用是可以将单个数据包装成数组，或者取出数组中第一个元素。

  • ‘ONE’：如果输入值是一个列表，获取列表的第一个元素，并将其设置为当前的 value，不支持其他类型。
  • ‘MANY’：如果输入值非列表，则创建一个列表，并将该输入值的首元素设置为列表中的元素；如果输入值为 null，则创建一个空列表。不支持列表类型。

modify

modify又分成如下模式：

  • modify-overwrite-beta： 模式是对指定的key的数据进行操作，无法key是否存在或已存在值，始终会对其进行操作
  • modify-default-beta ：当键值对应的值是null时写入
  • modify-define-beta ：当键值不存在时写入
