设计模式——建造者模式

发表于 2023-07-29 分类于 Java ， 设计模式
本文字数： 1.8k

建造者模式是让使用者通过链式调用的方式构建自己需要的对象

建造者模式编写的类通常命名为 XxxBuilder ，再通过链式调用它的方法来构建对象。

通过建造者模式，我们就可以有选择的去填充对象当中的属性，构造自己需要的对象。
