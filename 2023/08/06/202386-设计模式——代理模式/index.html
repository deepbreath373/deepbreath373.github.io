设计模式——代理模式

发表于 2023-08-06 分类于 Java ， 设计模式
本文字数： 2.7k

代理模式分为静态代理和动态代理。

代理

代理是什么？让代理对象来控制某对象的访问，这个过程就是代理对象代理了这个对象。如果想要访问该对象，必须要通过调用这个代理对象。通常，也用来隐藏具体实现类的实现细节，调用者不需要知道具体的调用过程，一些交由代理对象完成。当然，具体的实现还是由原对象完成。

代理方式分为静态代理和动态代理。

静态代理

静态代理就是在程序运行之前，由开发者手动编写相应的代理类，这样在程序执行的时候，就已经完成了代理类的编译。
